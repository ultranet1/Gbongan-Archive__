# Gbongan Archive - Full Stack Application

## Project Overview

Gbongan Archive is a comprehensive community archive platform built with Django (backend) and React (frontend). The application preserves and showcases the history, culture, people, and heritage of Gbongan community.

www.gbonganarchive.com.ng

## Features

### Core Features
1. **Archive Management**: Browse and search historical archive items organized by categories
2. **People Directory**: Database of notable people from Gbongan with biographical information
3. **Location Mapping**: Important historical and cultural locations with coordinates
4. **News & Events**: Community news and event announcements
5. **Cultural Content**: Oriki (praise poetry), Anthem, History, Heritage information
6. **User Contributions**: Community members can submit archive items, news, or contact messages

### Admin Features
1. **Content Moderation**: Review and approve user contributions
2. **Archive Management**: Create, edit, and manage all archive content
3. **User Management**: Django admin interface for full system control
4. **Media Management**: Upload and manage images and files
